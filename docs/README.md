# 📚 Tài Liệu Hướng Dẫn Dự Án LampNis

Chào mừng bạn đến với thư mục tài liệu `docs/` của dự án **LampNis**! Thư mục này chứa các tài liệu cần thiết để giúp bạn dễ dàng tìm hiểu, chỉnh sửa, và duy trì mã nguồn của dự án mà không cần phải ghi nhớ mọi chi tiết.

## 📁 Các Tài Liệu Hiện Có

Dưới đây là danh sách các tài liệu hướng dẫn chi tiết đang có trong thư mục này:

### 1. [Hướng Dẫn Chỉnh Sửa (HUONG_DAN_CHINH_SUA.md)](./HUONG_DAN_CHINH_SUA.md)
Đây là tài liệu quan trọng nhất nếu bạn đang muốn thay đổi nội dung, màu sắc, hay thêm các route/dự án con vào trong **LampNis**.
Tài liệu bao gồm:
- **Cấu trúc dự án tổng quan**: Hiểu rõ thư mục nào làm nhiệm vụ gì.
- **Tùy chỉnh Login, Mã PIN & Màu đèn**: Cách cấu hình mã bí mật (master, access, v.v.) và thiết lập màu sắc tương ứng.
- **Thêm dự án/trang mới**: Hướng dẫn từng bước từ việc tạo file Vue mới đến đăng ký route trong `src/router/index.js`.
- **Đổi theme, màu sắc & văn bản**: Bảng mã màu hex chuẩn và cách áp dụng nhanh chóng qua CSS Variables.
- **Troubleshooting**: Cách khắc phục nhanh các lỗi phổ biến (trắng trang, animation giật, v.v.).

---

## 🚀 Cách Sử Dụng Tài Liệu

- **Đọc lướt (Skimming)**: Hãy sử dụng mục lục ở đầu mỗi tài liệu để đi thẳng đến phần bạn cần chỉnh sửa.
- **Copy-Paste**: Các đoạn code trong tài liệu đã được chuẩn hóa. Bạn chỉ cần copy và đổi thông số theo ý muốn.
- **Cập nhật**: Khi bạn thêm một tính năng lớn hoặc thay đổi luồng của dự án, hãy nhớ cập nhật lại các file markdown này để người tiếp theo (hoặc chính bạn sau này) dễ dàng theo dõi!
